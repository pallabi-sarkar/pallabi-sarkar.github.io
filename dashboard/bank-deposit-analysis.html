Bank Deposit Performance Analysis

Excel Dashboard Case Study

Project Overview

This project analyzes bank transaction data to understand deposit performance across branches, account types, and customer segments. An interactive Excel dashboard was built to identify revenue drivers and potential growth opportunities.

Dataset Overview

Column Description
Date Transaction date
Branch Bank branch handling the transaction
Account Type CD, Savings, Checking, IRA
Amount Deposit amount
Customer Type New or Existing customer
Opened By Teller or New Accounts team

Total Deposits Analyzed: 6,535,772

Tools Used

  • Microsoft Excel
  • Pivot Tables
  • Pivot Charts
  • Data Visualization
  • Interactive Dashboard

Key Business Questions

  • What is the total deposit amount?
  • Which account type generates the highest deposits?
  • Which branch contributes the most revenue?
  • Do new customers bring more deposits?
  • Which team generates more revenue?
  • Which branch should be prioritized for growth?

Dashboard Preview

Key Insights

  • Central branch generates the highest total deposits.
  • CD accounts contribute the largest share of deposits.
  • Existing customers generate more revenue than new customers.
  • The New Accounts team contributes higher deposit value than Teller transactions.
  • Westside branch handles many transactions but lower deposit value.

Business Recommendations

  • Promote CD accounts across branches to increase deposit value.
  • Focus marketing on high-value new customer acquisition.
  • Improve cross-selling strategies at Westside branch.
  • Expand New Accounts team strategy for higher revenue.
