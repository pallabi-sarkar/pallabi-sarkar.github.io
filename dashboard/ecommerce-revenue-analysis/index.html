E-commerce Revenue Analysis Dashboard

Project Overview

This project analyzes e-commerce order data to understand revenue performance, regional trends, and product demand. An interactive Excel dashboard was created to visualize key metrics such as total revenue, sales trends, and top-performing categories.

Dataset

The dataset contains order-level information including order date, category, region, quantity sold, and revenue.

Dashboard

Key Insights

  • Technology category generated the highest revenue.
  • West region contributed the most sales.
  • Sales showed strong growth in the last quarter.
  • Some sub-categories dominate revenue contribution.

Tools Used

  • Microsoft Excel
  • Pivot Tables
  • Interactive Dashboard
